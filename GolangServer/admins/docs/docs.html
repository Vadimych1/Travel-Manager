About

It`s an HTTP API server for mobile app Travel Manager. For start run:

go run .

or

./start.bat

Server will start at port %port. You also can change it in config.yml.

Usage

An API includes the following:

  1. Travel plans:
  2. Get all travel plans localhost:%port/get_all_travels
  3. Get travel plan localhost:%port/get_travel (not competely implemented)
  4. Create travel plan localhost:%port/create_travel
  5. Update travel plan localhost:%port/update_travel (not competely implemented)
  6. Delete travel plan localhost:%port/delete_travel (not competely implemented)
  7. Users:
  8. Register localhost:%port/register
  9. Login localhost:%port/login
  10. Update user localhost:%port/update_user (not competely implemented)
  11. Delete user localhost:%port/delete_user (not competely implemented)

Server interface

The server have the interface for admins which on the localhost:%port/admins (or on the port which you can change in config.yml). It has the following functions:

  1. Database info:
  2. Users
  3. Travel plans
  4. SQL requests into DB
  5. Server info:
  6. Server version
  7. Server uptime
  8. Debug info
